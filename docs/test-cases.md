## 3. PDF 다운로드 테스트

### 3.1 직접 URL 다운로드
```
테스트: "https://arxiv.org/pdf/1706.03762.pdf 다운로드해줘"
예상결과: PDF 다운로드 성공, 파일명과 메타데이터 표시
```

### 3.2 검색 후 다운로드
```
테스트1: "BERT 논문 검색해줘"
테스트2: "첫 번째 논문 다운로드해줘"
예상결과: 검색 결과의 첫 번째 PDF 다운로드
```

### 3.3 중복 다운로드 시도
```
테스트: "https://arxiv.org/pdf/1706.03762.pdf 다시 다운로드"
예상결과: "이미 다운로드됨" 메시지와 캐시 정보
```

### 3.4 잘못된 URL
```
테스트: "https://example.com/notapdf.html 다운로드"
예상결과: PDF가 아닌 파일 오류 메시지
```

### 3.5 큰 파일 다운로드
```
테스트: "[50MB 이상 PDF URL] 다운로드"
예상결과: 파일 크기 제한 오류 메시지
```

## 4. PDF 질의응답 테스트

### 4.1 최근 PDF에 질문 (파일명 없이)
```
테스트: "이 논문의 핵심 contribution이 뭐야?"
예상결과: 가장 최근 다운로드한 PDF에서 답변
```

### 4.2 파일명으로 지정
```
테스트: "1706.03762.pdf에서 attention mechanism 설명해줘"
예상결과: 특정 파일에서 관련 내용 검색
```

### 4.3 URL로 지정
```
테스트: "https://arxiv.org/pdf/1706.03762.pdf에서 limitation 찾아줘"
예상결과: URL로 지정한 PDF에서 검색
```

### 4.4 다운로드 안 된 파일 질문
```
테스트: "unknown.pdf에서 methodology 설명해줘"
예상결과: "PDF를 먼저 다운로드하세요" 오류
```

### 4.5 한국어 질문
```
테스트: "이 논문의 실험 결과가 어때?"
예상결과: experimental results 관련 내용 검색
```
