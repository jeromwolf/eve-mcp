# 알려진 문제점 및 제한사항

## 현재 제한사항

### 1. PDF 텍스트 추출
- **문제**: 일부 PDF는 텍스트 추출이 안 됨
- **영향**: 스캔된 PDF, 이미지 기반 PDF는 질의응답 불가
- **해결책**: OCR 기능 추가 검토 중

### 2. 메모리 관리
- **문제**: PDF 캐시가 무한정 증가
- **영향**: 장시간 사용 시 메모리 부족 가능
- **임시해결**: Claude Desktop 재시작
- **계획**: LRU 캐시 구현 예정

### 3. 질의응답 정확도
- **현재**: 단순 키워드 매칭
- **한계**: 
  - 동의어 인식 불가 (예: "주의 메커니즘" vs "attention mechanism")
  - 문맥 이해 부족
  - 긴 문장 검색 어려움
- **개선 계획**: 임베딩 기반 검색 구현

### 4. 보안 고려사항
- **URL 검증**: 기본적인 검증만 수행
- **권장사항**: 신뢰할 수 있는 학술 사이트만 사용
- **금지**: 
  - 개인정보가 포함된 PDF
  - 비공개 문서
  - 의심스러운 출처

### 5. 플랫폼별 이슈

#### Windows
- 경로에 한글/특수문자 있으면 오류 가능
- 백슬래시(`\`) 경로 구분자 주의

#### macOS
- 첫 실행 시 보안 경고 가능 → "허용" 선택
- M1/M2 Mac에서 Node.js 버전 확인 필요

## 임시 해결 방법

### PDF 텍스트 추출 실패 시
1. 다른 소스에서 PDF 다시 다운로드
2. PDF를 다시 생성 (가능한 경우)
3. 텍스트 기반 PDF 사용

### 메모리 문제 발생 시
1. Claude Desktop 재시작
2. 불필요한 PDF 다운로드 자제
3. 주기적으로 재시작

### 질의응답 개선
1. 구체적인 키워드 사용
2. 영어 용어 사용 (한글보다 정확)
3. 짧고 명확한 질문
